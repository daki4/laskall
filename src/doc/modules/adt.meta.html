BuildAdt (typename)
Creates a new Algebraic Data Type (ADT) factory. Similar to Rust's enum implementation, but for Lua. Creates a base type that can be extended with variants through Register() and whose instances are created through AdtCtor(). name: string, baseState: { [string]: function }, states: VariantDef[], Register: fun(self: table, func: function, name: string, takesValue: boolean): nil, AdtCtor: fun(self: table, state: function, value: any?): table }
